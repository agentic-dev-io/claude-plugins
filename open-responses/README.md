# Open Responses

Open Responses specification implementation for multi-provider LLM orchestration.

## Skills

| Skill | Description |
|-------|-------------|
| open-responses-orchestrator | Multi-provider routing and streaming |
| agents-sdk-open-responses | OpenAI Agents SDK with local models |
| rust-open-responses-engine | High-performance Rust backend |

## Features

- Multi-provider routing (OpenAI, Anthropic, Ollama, LiteLLM)
- Streaming support with SSE
- Tool calling and structured outputs
- Anthropic Agents SDK integration
- Rust backend with Axum/Burn

## Installation

```bash
cc --plugin-dir ./open-responses
```
