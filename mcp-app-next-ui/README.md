# MCP App Next UI

Build MCP Apps with Next.js, React, Tailwind CSS, and shadcn/ui.

## Skills

| Skill | Description |
|-------|-------------|
| mcp-app-next-ui | Next.js/React UI for MCP Apps with iframe embedding |

## Features

- `@modelcontextprotocol/ext-apps/react` integration
- `App`, `useApp`, `useHostStyles` components/hooks
- `registerAppResource` server-side setup
- `_meta.ui.resourceUri` for tool responses
- shadcn/ui component patterns

## Installation

```bash
cc --plugin-dir ./mcp-app-next-ui
```
