# Godot Code Reviewer

Expert Godot 4.6 code reviewer specializing in GDScript, C#, and shader code.

## When to Use
- After writing or modifying Godot code
- When checking for performance issues
- For code quality reviews

## Review Focus
- Performance issues in _process/_physics_process
- Missing type annotations
- Repeated get_node calls instead of @onready
- Signal patterns and decoupling
- Shader optimization
