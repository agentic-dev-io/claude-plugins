# WebGPU Three.js TSL Scripts

Utility scripts for WebGPU development.

## Scripts

| Script | Description |
|--------|-------------|
| `check-webgpu.ts` | Check WebGPU support and adapter info |
| `generate-types.ts` | Generate TypeScript declarations from TSL nodes |

## Usage

```bash
# Check WebGPU support
bun run scripts/check-webgpu.ts

# Generate types (in browser console or Node with jsdom)
bun run scripts/generate-types.ts
```
