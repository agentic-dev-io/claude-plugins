# Frontend Design Scripts

Utility scripts for frontend development workflow.

## Scripts

| Script | Description |
|--------|-------------|
| `setup-project.sh` | Initialize Next.js project with Tailwind and shadcn/ui |
| `add-components.sh` | Batch add shadcn/ui components |

## Usage

```bash
# Setup new project
bash scripts/setup-project.sh my-app

# Add common components
bash scripts/add-components.sh
```
